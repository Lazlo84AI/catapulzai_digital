## Qui sommes-nous ?

**CatapulZ AI** est une société française de conseil, développement et venture building
basée sur l'intelligence artificielle. Elle accompagne entrepreneurs, PME et executives
à tirer des revenus réels et mesurables de l'IA via des systèmes agentiques et du vibe coding.

**Fondateur / CTO :** rôle de développeur web senior (20 ans XP) + ingénieur IA
**Spécialités :** React, TypeScript, Tailwind, Supabase, N8N, Dust

## Value Proposition

## Stack Technique

- **Frontend :** React + TypeScript + Tailwind CSS
- **Backend/BDD :** Supabase (auth, database, storage)
- **Déploiement :** Local → Vercel / Netlify → URL cible (domaine existant)
- **Agents IA :** N8N + Dust (Phase 2)
- **Vibe coding :** approche rapide, pragmatique, production-grade
